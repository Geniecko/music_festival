## Music Festival

### Technologies used

- HTML
- CSS(SASS & BEM)
- JavaScript
- Fetch API
- SplideJs
- Webpack
- Figma

### Setup

To run this project, install it locally using npm:

- Run ```npm install``` to install packages
- Run ```npm run prod``` to build js and css for production or run ```npm run dev``` for development
